Trabalho final da matéria de Teoria da Computação ministrada pelo professor Islame Felipe do Curso de Bacheralado em Ciência da Computação.

Equipe:
- João Roberto
- Caio Mello
- Vinícius

Para essa atividade primeiro desenvolvemos os autômatos usando um _fork_ - versão modificada - que desenvolvemos do programa alemão [FLACI](https://github.com/gitmh/FLACI).

Nesse repositório, além do documento final para avaliação pelo professor também deixamos os dois arquivos .json que contém as máquinas de turing que construímos, caso deseje rodar elas no simulador, baixe nossa versão [aqui](https://github.com/jrport/FLACI). Vá em `Abstract Automata`, clique no canto superior direito, no ícone do arquivo, então selecione o .json que deseje carregar no simulador e então rode uma computação!
